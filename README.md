# powerpoint-2-interactiveboard
Converts PowerPoint files (.pptx) into basic InteractiveBoard (plugin) scene configurations, allowing easy button positioning and link scene configuration.

Run that in a python notebook like Google Colab or Jupyter.

Make sure powerpoints aspect ratios match the amount of maps it will use.

When ran, it will ask you to upload the powerpoint file with the necessary font files if necessary (I used installed minecraft fonts, so I needed to upload them as well).

And also the size in maps, my example is 5 maps wide x 3 maps tall. Again, make sure PPT's aspect ratio matches this.

You can set other properties such as picture/background prefix, file name and button properties, and if you want the picture to use dithering.
